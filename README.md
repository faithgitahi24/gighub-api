# GigHub Nairobi Freelance Gigs API

Admission Number: C027-01-0889/2024

## description

This API manages gig listings for freelancers and clients.

Features:
- List all gigs with optional filtering by category and budget range
- Retrieve a specific gig by ID
- Search gigs by title
- Create new gigs with validation
- Update a gig's budget or status
- Delete gigs


## Run the Project

Install the required packages:

```bash
pip install fastapi uvicorn
```

Run the server:

```bash
uv run uvicorn main:app --reload
```

3. Open Swagger UI:

```
http://127.0.0.1:8000/docs
```
